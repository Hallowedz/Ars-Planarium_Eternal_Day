Planarium Eternal Day 1.0.0
Minecraft 1.21.1 / NeoForge 21.1.x / Ars Nouveau

Purpose
- Replaces the Ars Nouveau Planarium dimension's black void effect with the vanilla Overworld sky.
- Enables skylight.
- Locks celestial time at 6000 ticks (permanent noon).
- Changes the Planarium biome's black sky and fog colors to daylight colors.

Installation
- Install this jar in the server mods folder for dedicated servers.
- Install it in the client mods folder as well for consistent modpack distribution.
- For singleplayer, install it in the client's mods folder.
- Keep Ars Nouveau installed.

Scope
- Overrides only:
  data/ars_nouveau/dimension_type/jar.json
  data/ars_nouveau/worldgen/biome/jar.json
